Metodología

Descripción de las etapas del proyecto

La investigación propuesta tiene un enfoque aplicado, orientado a ofrecer soluciones prácticas para los objetivos delineados en este estudio. Se caracteriza por su naturaleza descriptiva y analítica que permite detallar las características y propiedades del objeto de estudio. Se incluyen variables geoespaciales y datos relevantes para la zona de investigación. Además, se empleará una metodología correlacional para evaluar el grado de relación entre los aspectos analizados y los riesgos identificados.

Este trabajo implica un proceso integral que comprende la investigación, recopilación y estandarización de bases de datos de las ASADAS, acueductos rurales y de las cuatro municipalidades dentro de la zona de estudio. Se verifica la cobertura de sistemas, las fuentes de recurso hídrico como nacientes y los componentes relacionados tanto con la distribución como con la conducción de redes. En muchos casos, se generará información nueva con hojas electrónicas y archivos vectoriales para el análisis. Estos datos alfanuméricos constituyen la base de los mapas finales.

La cartografía y la construcción de la base de datos se llevarán a cabo utilizando un Sistema de Información Geográfica Libre. Los datos recopilados de las ASADAS, acueductos rurales y municipalidades serán estandarizados para su integración en un visor y estarán disponibles para descarga y manejo a través de una sección dedicada en la página web, así como para su utilización en un SIG de escritorio.
